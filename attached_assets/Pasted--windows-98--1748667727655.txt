اريد اسوي موقع اليه شخصي اعرض بي اعمالي
يكون الموقع عبارة عن windows 98 نفس النظام ضبط تكدر تستخدم مكاتبهم ايضًا

الاقسام الي احتاجها

🪪 1. Who am I?
📝 نافذة باسم “Me.txt”

📁 2. My Projects
🗂️ نافذة تشبه File Explorer
كل مشروع يظهر كأيقونة فولدر، تضغط عليه يفتح نافذة فيها:
 • صورة المشروع أو فيديو
 • وصف مختصر
 • رابط ويبسايت

🎵 3. Music
🎧 نافذة مشغل MP3 (Winamp-style)
 • بلايليست باسم: weedy_playlist.m3u
 • روابط مدمجة من ساوندكلاود / سبوتيفاي
 • أزرار: Play / Pause / Next
 • موسيقى Chill أو Lo-Fi أو Oud retro

📸 4. Gallery
🖼️ نافذة عارض صور قديم (Windows Photo Viewer)
 • صور شخصية، صور مشاريع، أو حتى AI art
 • أزرار Next/Prev
 • تأثير عرض شرائح تلقائي

🧠 5. Thoughts / Blog
📝 نافذة باسم “ideas.notepad”
 • مقالات قصيرة أو “خربشات”
 • خواطر عن تصميم، وقت، تعب، شغل

📦 6. Downloads
💾 نافذة فيها روابط تحميل
 • سيرتي الذاتية (.pdf)
 • صور جاهزة (zip)
 • عروض مشاريع
 • زر “Download Now”


💬 7. Contact
📫 نافذة شكلها مثل “Send Email”
 • نموذج فيه:
 • Name
 • Email
 • Message box
• زر: “Send”

🎮 8. Terminal Fun / About Me (CLI style)
🖥️ نافذة Terminal وهمية، تكتب أوتوماتيكياً


عناصر وتفاصيل من نظام Windows القديم ممكن تستخدمها

🟦 1. زر Start
زر بأسفل الشاشة، لما تضغط عليه، تطلع قائمة فيها:
 • “My Computer” → يفتح مجلد مشاريعك
 • “Run” → يفتح نافذة تدخل بيها أمر مثل whoami
 • “Shut down” → يعطيك نافذة وهمية بيها “Bye 👋”

🪟 2. النوافذ القابلة للسحب والتكبير
 • كل قسم (projects, music, contact…) يكون نافذة قابلة للتحريك (drag)
 • فيها زر تصغير، تكبير، إغلاق تمامًا مثل نوافذ ويندوز 98

🖥️ 3. سطح المكتب (Desktop)
 • خلي أقسام الموقع على شكل أيقونات على سطح المكتب مثل

📂 4. File Explorer وهمي
 • مثل شجرة ملفات (tree view) فيها

🧾 5. رسائل منبثقة (Pop-ups)
مثل نوافذ الخطأ القديمة
[!] Are you sure you want to leave this page?
[Yes]   [No]

⌛ 6. Loader كلاسيكي
استخدم شريط تحميل كلاسيكي أو الساعة الرملية القديمة 🕳

📅 7. شريط المهام (Taskbar)
بأسفل الشاشة:
 • الوقت (يتحدث تلقائيًا)
 • أيقونات النوافذ المفتوحة
 • صوت بدء النظام عند أول زيارة

🗑️ 8. Recycle Bin (سلة المحذوفات)
تكون أيقونة فارغة، لما تدخلها تطلع رسالة:
“No regrets here — just past versions of you.”


🖼️ 9. خلفية كلاسيكية
 • مثل خلفية Windows 98 لكن بنسخة معدلة عليها شعارك أو مشروعك.
